FORRETT

Bygging av JavaScript-prosjekter med Grunt

En moderne nettside, nesten uansett hvor enkel den måtte være, må igjennom en rekke steg før den er klar for verdensveven. Sammenslåing av kildefiler, minifisering, optimalisering av bilder, erstatning av variabler brukt under utvikling, og selvfølgelig kjøring av tester og kodeanalyser.
Grunt er et byggeverktøy skrevet i JavaScript basert på en fornuftig plugin-arkitektur som løser ethvert problem du måtte kaste på den. I tillegg er den ekstremt enkel å utvide.

Varighet: 10-30 min

Funksjonelle tester med CasperJS og ResurrectIO

For filmtjenesten Comoyo View ble det laget et sett med enkle funksjonelle tester for å være sikker på at innloggingsfunksjonen på siden alltid fungerte etter hver deploy. Chrome pluginen resurrectIO kan generere JavaScript-koden nødvendig for å kjøre en test. Denne koden er enkelt forståelig og kan endres og vedlikeholdes manuelt. Testene ble kjørt etter hver deployment av Jenkins og en hodeløs browser (PhantomJS).

HOVEDRETT

Java 8: Return of the Java

Java som plattform er sterkere enn noen gang, men språkets alder begynner å vise seg. Den siste store nyheten var generics for 10 år siden. Dette er i ferd med å endre seg. Versjon 8 gir oss de største nyvinningene Java har sett.
Java 8 åpner endelig porten til det forjettede land av funksjonell programmering! Lær om superkreftene språket gir oss utviklere gjennom nye features som Lambdaer, Extension-metoder og Streams. Vi trekker også noen sammenligninger til alternative språk på JVMen som Scala og Clojure.

DESSERT

Clojure spiser XMLen din til frokost

Hvordan man enkelt kan lese og manipulere XML-filer i Clojure ved hjelp av Clojures implementasjon av Zipperdatastrukturen.
XML er ikke alltid enkelt å prosessere, spesielt ved ulogiske strukturer og manglende skjemadefinisjoner. I overgangen fra Java til Clojure må man fortsatt behandle mye XML. Jeg skal vise hvordan man enkelt kan lese og manipulere XML-filer i Clojure ved hjelp av Clojures implementasjon av Zipperdatastrukturen. Ved bruk av disse metodene vil behandling av XML data gå lekende lett.
