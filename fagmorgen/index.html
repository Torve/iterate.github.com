How to Build, Ship and Run Apps with Docker

Docker er et alternativ til å kjøre applikasjoner rett på serveren og virtuelle maskiner. Docker isolerer prosesser og applikasjoner. Når man er inne i et docker image ser man i praksis ikke resten av systemet, og individuelle docker prosesser kan gjøre som de vil uten å skape problemer for andre prosesser (nesten iallefall). I motsetning til tradisjonelle virtuelle maskiner krever de mye mindre ressurser da de ikke emulerer maskinvare, men bruker verktøy som finnes i Linux fra før.

På seminaret kan du lære om docker og dokku, hvordan det brukes og hvordan vi har satt det opp internt i Iterate
