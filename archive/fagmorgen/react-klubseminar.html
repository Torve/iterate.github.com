Klubbseminar: Getting started with React.js

Bli med på klubbseminaret og lær om React. Kaffe, te og enkel frokost vil bli servert.

Facebook har virkelig begynt å feste seg som en av de ledende aktørene innen for frontend-rammeverk, der React har vært spydspissen. Det er nå blitt et helt økosystem av biblioteker som gjør livet for en frontend utvikler mye enklere. Utbyttet er et innblikk i et interessant og nyskapende rammeverk innen webutvikling, nemlig React.js. Man får også en introduksjon til noen deler av React.js-økosystemet.

Seminaret passer best til: Web- og mobilutviklere, både frontend og fullstack

Nivå:Nybegynner

Lyntaler:

Introduksjon til React - Magnús Dæhlen, systemutvikler

Magnús skal gi en introduksjon til React, hvordan du tar det i bruk og dele erfaringer rundt bruken av React.

React Native - Truls Skeie, systemutvikler

Du får en introduksjon til React-Native. React-Native gjør det mulig å skrive mobil-apps med ReactJS og hvordan kan man overføre kompetansen og erfaringen fra React til React Native. Foredraget forutsetter basic React-kunnskap.

GraphQL - Henrik Glasø Skifjeld, systemutvikler

Facebook har nylig beskrevet en ny måte å utveksle data mellom klient og server som et alternativ til REST API - nemlig noe som kalles GraphQL. Hva er det? Hvordan fungerer det? Hvilke problemer løser det?
